# closure-np-cljs

"Building an Application with Closure" tutorial.

## Usage

### Steps

* Open a shell.
* Navigate to project directory.
* Execute `lein figwheel` to start (development) server.
* When prompted, open a browser and open `localhost:3349/notepad.html`.

Enjoy!
